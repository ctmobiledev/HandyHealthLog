HANDY HEALTH LOG is a simple app for recording your personal health care events. It's a lot easier jotting them in this app than a spiral notebook, and a lot easier to carry around, too!

To begin, tap the Add New Event button. The Data Type spinner lets you choose from one of the following data event types:

BP Blood Pressure Reading. Enter your systolic and diastolic numbers and your pulse in beats per minute.
EXERC Exercise. You can enter the number of minutes you were active, or your activity in miles or laps.
GLU Glucose/Blood Sugar Reading. Enter your blood sugar count.
OTHER Any other measures you wish to record. Use the Description 1 and Description 2 fields as reminders.
RX Prescription. For registering when you took a dose of a prescription medication.
TEMP Body Temperature Reading. Enter your temperature in degrees F.
WGT Weight Reading. Enter your weight in pounds and ounces.

At any time, you may export all entered data to a CSV file for import to spreadsheet applications such as Microsoft Excel (tm) or LibreOffice Calc. Tap the Export Events to CSV File button. The resulting file, HealthData.CSV, contains all entries you've made to date. Entries are sorted in chronological order.

To access this file, you may either (a) use an Android-based file manager app to search for the "HHL" folder, or (b) attach your Android device with the USB data/recharge cable and use your computer's file manager to search for the "HHL" folder.

You may also import entries from a CSV file you've copied to your Android device. To do this, tap the Import Events from CSV File button. You must have a file called ImportHealthData.CSV in the HHL folder. To get a feel for the format such a file should take enter ONE record using Add New Event and then tap the Export Events to CSV File button.

The first line in the file has headers describing each of the values, and are separated by the vertical bar (|) character.

All successive lines contain actual data, and values are separated by a vertical bar (|) character. There is no trailing vertical bar after the Description 2 data value.

NOTE: An import operation will DELETE ALL EXISTING DATA. You'll be asked to confirm this operation before you proceed with it.
